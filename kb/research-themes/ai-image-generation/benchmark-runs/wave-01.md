# Wave 01: ベンチマーク実行結果

実行日: 2026-03-30

## 実行条件

| 項目 | 値 |
|------|-----|
| モデル | NB2 (gemini-2.5-flash-image), NB Pro (gemini-3-pro-image-preview), GPT Image 1.5 Medium |
| 題材数 | 3（テック/ライフスタイル/金融） |
| 生成枚数 | 9枚（3モデル×3題材） |
| アスペクト比 | 16:9（NB2/Pro）、1536x1024（GPT Image） |

- **GPT Image 1.5**: 文字完璧。脳アイコン+回路基板。レイアウトが洗練されていて広告的。16:9を正しく反映

## 用途別推奨（多様性は善）

| 用途 | 推奨モデル | 理由 |
|------|----------|------|
| **サムネ量産（速度+品質バランス）** | NB2 | 6-9秒で高品質。コスパ良好 |
| **テキスト入りバナー** | NB2 or NB Pro | 文字レンダリングが正確。NB Proは16:9が安定 |
| **フォトリアル差込画像** | NB2 | lifestyle題材で最もリアルな写真品質 |
| **デザイン/イラスト風** | GPT Image 1.5 | フラットデザイン、広告的レイアウトに強い |
| **バナー/ヘッダー（16:9確実）** | NB Pro | アスペクト比制御が最も正確 |

## 発見と知見

1. **NB2がWave 01の総合最強**: 速度(3倍速)、品質(最高点)、テキスト(正確)の三拍子。コストも$0.067で手頃
2. **NB Proはアスペクト比制御が強み**: 16:9指定を最も忠実に再現。バナー用途で有利
3. **GPT Image 1.5はデザイン的**: イラスト/フラットデザイン風の仕上がり。写真リアリズムではGemini系に劣る
4. **テキストレンダリング**: 3モデルとも良好。大差なし
5. **NB2の正方形問題**: 16:9指定でも正方形になることがある。API側のアスペクト比パラメータ指定方法を要調査
